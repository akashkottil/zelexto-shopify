SECTION 1 — PRODUCT OVERVIEW
=====================================================================

1.1 THEME GOAL

The goal of this project is to build a premium, conversion-optimized, reusable Shopify theme that:

• Maximizes ecommerce conversion rates
• Is fully compatible with Shopify Online Store 2.0
• Requires zero third-party apps for essential functionality
• Supports advanced features via custom apps developed by our platform
• Can be sold on the Shopify Theme Store and private marketplaces
• Serves as the base theme for our own ecommerce stores

The theme must prioritize:

• Performance
• Conversion optimization
• Scalability
• Modular architecture
• Merchant usability

The theme should achieve:

• PageSpeed score above 90
• Lighthouse performance above 90
• Minimal JavaScript dependency
• Mobile-first performance

---------------------------------------------------------------------

1.2 TARGET USERS

Primary users:

Shopify merchants who want a high-converting store without installing many apps.

Examples:

• DTC brands
• Fashion stores
• Beauty brands
• Electronics stores
• Lifestyle products
• Home decor
• Dropshipping businesses
• Subscription brands

Merchant skill level:

• Beginner merchants
• Intermediate store owners
• Advanced Shopify developers

---------------------------------------------------------------------

1.3 TYPES OF STORES SUPPORTED

The theme must support:

• Single product stores
• Small catalog stores
• Large catalog stores
• Fashion brands
• Beauty brands
• Digital product stores
• Subscription stores
• Wholesale-ready stores

---------------------------------------------------------------------

1.4 CONVERSION-FIRST DESIGN PHILOSOPHY

Every component must be designed around conversion rate optimization.

Core CRO principles:

• Clear CTAs

• Scarcity messaging

1.5 SHOPIFY COMPATIBILITY

Theme must support:

---------------------------------------------------------------------

The theme must be:

All UI components must be built as reusable sections or snippets.

---------------------------------------------------------------------

2.3 FOLDER STRUCTURE

/theme-root
|
|-- layout
|-- templates
|-- sections
|-- snippets
|-- assets
|-- config
|-- locales

---------------------------------------------------------------------

2.4 FOLDER PURPOSES

/layout

Contains the main layout files.

Example:

theme.liquid

Purpose:

Defines the HTML skeleton of the site including:

• head
• header
• footer
• scripts

---------------------------------------------------------------------

/templates

Contains page templates.

Examples:

product.json
collection.json
page.json
blog.json
article.json
cart.json
search.json
404.json

---------------------------------------------------------------------

/sections

Reusable dynamic sections.

Examples:

hero-banner.liquid
featured-products.liquid
product-gallery.liquid
product-recommendations.liquid
testimonials.liquid

Sections must support dynamic configuration via JSON schema.

---------------------------------------------------------------------

/snippets

Reusable components.

Examples:

product-card.liquid
price.liquid
variant-selector.liquid
icon.liquid
rating-stars.liquid

---------------------------------------------------------------------

/assets

Static resources.

Examples:

theme.js
theme.css
component-cart.js
component-product.js

Images and fonts also stored here.

---------------------------------------------------------------------

/config

Theme configuration.

Examples:

settings_schema.json
settings_data.json

---------------------------------------------------------------------

/locales

Translation support.

Example:

en.default.json
